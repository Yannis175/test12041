## XWiki 简介

XWiki 是一个基于 Java 的开源企业级 Wiki 平台，旨在提供强大的协作和知识管理功能。

### 主要特性

- **强大的编辑功能**：支持 WYSIWYG（所见即所得）编辑器，以及功能丰富的 Wiki 语法
- **灵活的权限管理**：提供严格的权限控制系统，确保内容安全
- **丰富的扩展生态**：提供超过 900 个扩展，包括应用程序、宏、皮肤、插件和主题等
- **多语言脚本支持**：支持在 Wiki 界面中执行服务器端脚本，如 Velocity、Groovy、Python、Ruby 和 PHP 等
- **结构化数据管理**：允许用户创建和管理结构化数据，满足复杂业务需求

### 应用场景

- 企业知识库
- 协作式内网
- 公共网站
- 业务应用开发

### 技术特点

- 基于 Java 开发
- 采用 LGPL 开源许可证
- 灵活的架构设计
- 支持多种部署方式

### 许可证

所有 XWiki 软件均采用 LGPL 开源许可证发布。
